## Block 1: Base Building (Mar 17 – Apr 13, 2026)

### Schedule

| Day | Session |
|-----|---------|
| Mon | Lift A — Vertical Power (heavy: OHP, Chin-ups, Leg Press, Bulgarians) |
| Tue | Run — Walk/run intervals |
| Wed | Lift B — Width & Arms (moderate: Pull-ups, Dips, Laterals, Arms) |
| Thu | Run — Walk/run intervals |
| Fri | Lift C — Horizontal Strength (mixed: Incline Bench, RDL, Cable Rows) |
| Sat | Run — Walk/run intervals (longest session of the week) |
| Sun | Full rest |

### Lifting Program

**Day A: Vertical Power (~65 min, 15 working sets)**
1. Hanging Leg Raises — 2 x 10-12 (tempo 2-0-1-1, rest 60s)
2. Leg Press — 3 x 6-8 (tempo 3-1-1-0, rest 3m)
3. Bulgarian Split Squats — 2 x 8/leg (tempo 2-0-1-0, rest 60s alt legs)
4. Standing OHP — 3 x 6-8 (tempo 2-0-X-0, rest 3m)
5. Weighted Chin-Ups — 3 x 6-8 (tempo 2-1-X-1, rest 3m)
6. Standing Calf Raises — 2 x 8-10 (tempo 2-2-1-2, rest 90s)

**Day B: Width & Arms (~55 min, 14 working sets)**
1. L-Sit Progression Holds — 3 x 15-20s (isometric, rest 60s)
2. Pull-Ups / Lat Pulldowns — 3 x 8-10 (tempo 2-1-1-1, rest 2m)
3. Weighted Dips (Upright) — 3 x 6-10 (tempo 3-0-1-0, rest 2m)
4. Lateral Raises — 3 x 10-12 (tempo 2-0-1-1, rest 60s)
5. Face Pulls — 2 x 12-15 (tempo 2-1-1-2, rest 60s)
6. Barbell Curls + Rope Pushdowns — 3 x 10-12 each (superset, rest 90s)

**Day C: Horizontal Strength (~70 min, 17 working sets)**
1. Ab Wheel Rollouts — 2 x 8-10 (tempo 3-1-1-0, rest 60s)
2. Trap Bar RDL — 3 x 5 (tempo 3-1-X-1, rest 3m)
3. Incline Bench Press — 4 x 6-8 (tempo 3-1-1-0, rest 3m)
4. Seated Cable Rows — 3 x 10-12 (tempo 2-0-1-1, rest 2m)
5. Barbell Hip Thrusts — 3 x 8-10 (tempo 2-0-1-2, rest 2m)
6. Seated Calf Raises — 2 x 10-15 (tempo 2-2-1-2, rest 90s)

**Cooldown (every session):** Dead Hangs 2-3 x 30-45s

**Progression rule:** Hit top of rep range on ALL sets → add weight next session (2.5 kg upper, 5 kg lower)

**All runs at Zone 2 / conversational pace.** If you can't hold a conversation, slow down.

**Weekly running volume:**
- Week 1: ~110 min total
- Week 2: ~110 min total
- Week 3: ~115 min total
- Week 4: ~80 min total (deload)

### Week 4 Deload Protocol

**Lifting:**
- Same exercises, same weight
- Cut sets: 4→2, 3→2, 2→1
- Same rest periods, same tempo
- Should feel easy — that's the point

**Running:**
- Tue/Thu shortened to 25 min
- Saturday: 30 min walk only (no running)

- Carbs: 5-8 g/kg/day (do NOT go low-carb)
- Calories: at maintenance or slight surplus (+200-300 kcal)
- Creatine: 3-5 g/day
- Sleep: 8-9 hours

## Research Principles Guiding This Program

1. **Interference effect is real but manageable** — hypertrophy is least affected (Wilson 2012, Sabag 2018)
2. **Separate sessions by 6+ hours** when possible; lift first, run second (Baar 2014, Eddens 2018)
3. **80% of running is easy** (Zone 2) — minimizes AMPK activation and muscle interference (Seiler 2010)
4. **Deload every 3-4 weeks** — concurrent training accumulates fatigue from two sources (Pritchard 2015)
5. **Front-load hypertrophy** — push muscle building hardest in Block 1 when running volume is lowest
6. **Progressive overload on compounds** — heavy lifting improves running economy 2-8% (Beattie 2014)
7. **Protein 1.8-2.2 g/kg/day** — higher than lifting alone due to leucine oxidation from endurance work (Morton 2018)
8. **Never run in glycogen-depleted state** — amplifies interference effect (Hawley 2014)
9. **DUP across the week** — Day A heavy, Day B moderate, Day C mixed (Rhea 2002, Zourdos 2016)
10. **Sleep 8+ hours** — athletes <7hrs have 1.7x injury risk (Vitale 2019)
