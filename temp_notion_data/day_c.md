# FRI: Full Body C

## Warm Up

**Instructions:** Complete once in a circuit:

- **Bike or Rowing Machine** — 3-5 minutes - Zone 2 (120-150 heart rate)
- **Arm Circles** — 10 each side
- **Thread The Needle** — 10 each side
- **Band/Cable Face Pull** — 10
- **Band/Cable External & Internal Rotation (with arm by side)** — 10 each side
- **Walking Lunge** — 20 steps
- **Deep Squat Hold** — 60-90 seconds
- **Dead Hang** — 10 deep nasal breaths

## Workout: Full Body C

### Set 1: Complete the exercises back-to-back then rest for 2:00

- **1A: Barbell Strict Overhead Press**
  - Set 1: 6-10 (3 from failure)
  - Set 2: 6-10 (3 from failure)
  - Set 3: 6-10 (3 from failure)
- **1B: Pull-Ups (use assistance if needed)**
  - Set 1: 5-10
  - Set 2: 5-10
  - Set 3: 5-10

### Set 2: Complete the exercises back-to-back then rest for 2:30

- **2A: Single Leg Leg Press**
  - Set 1: 12-15 (2 from failure)
  - Set 2: 12-15 (2 from failure)
  - Set 3: 12-15 (2 from failure)
- **2B: DB RDL**
  - Set 1: 8-12 (2 from failure)
  - Set 2: 8-12 (2 from failure)
  - Set 3: 8-12 (2 from failure)

### Set 3: Complete the exercises back-to-back then rest for 2:00

- **3A: Dips**
  - Set 1: 8-12
  - Set 2: 8-12
  - Set 3: 8-12
- **3B: Unilateral Calf Raise (Weighted)**
  - Set 1: 6-10
  - Set 2: 6-10
  - Set 3: 6-10, 10 second rest, max reps, 10 second rest, max reps (same weight)
- **3C: DB Hammer Curl**
  - Set 1: 10-15
  - Set 2: 10-15
  - Set 3: 10-15, 10 second rest, max reps, 10 second rest, max reps (same weight)
- **3D: Hanging or Lying Leg Raises**
  - Set 1: AMRAP
  - Set 2: AMRAP
  - Set 3: AMRAP
