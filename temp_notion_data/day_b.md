# WED: Full Body B

## Warm Up

**Instructions:** Complete once in a circuit:

- **Bike or Rowing Machine** — 3-5 minutes - Zone 2 (120-150 heart rate)
- **Arm Circles** — 10 each side
- **Thread The Needle** — 10 each side
- **Band/Cable Face Pull** — 10
- **Band/Cable External & Internal Rotation (with arm by side)** — 10 each side
- **Walking Lunge** — 20 steps
- **Deep Squat Hold** — 60-90 seconds
- **Dead Hang** — 10 deep nasal breaths

## Workout: Full Body B

### Set 1: Complete the exercises back-to-back then rest for 2:00

- **1A: Deadlift**
  - Set 1: 6-8 (3 from failure)
  - Set 2: 6-8 (3 from failure)
  - Set 3: 6-8 (3 from failure)
- **1B: Band or Cable Woodchopper**
  - Set 1: 10
  - Set 2: 10
  - Set 3: 10

### Set 2: Complete the exercises back-to-back then rest for 2:00

- **2A: Incline DB Press**
  - Set 1: 6-10 (2 from failure)
  - Set 2: 6-10 (2 from failure)
  - Set 3: 6-10 (2 from failure)
- **2B: Prone Incline DB Row**
  - Set 1: 8-12 (2 from failure)
  - Set 2: 8-12 (2 from failure)
  - Set 3: 8-12 (2 from failure)

### Set 3: Complete the exercises back-to-back then rest for 1:30

- **3A: Quad Extension**
  - Set 1: 10-15
  - Set 2: 10-15
  - Set 3: 10-15, 10 seconds rest, drop weight then hit max reps
- **3B: Press Ups**
  - Set 1: AMRAP
  - Set 2: AMRAP
  - Set 3: AMRAP
- **3C: DB Curl**
  - Set 1: 10-15
  - Set 2: 10-15
  - Set 3: 10-15, 10 seconds rest, drop weight then hit max reps
- **3D: DB Lateral Raise**
  - Set 1: 10-15
  - Set 2: 10-15
  - Set 3: 10-15, 10 seconds rest, drop weight then hit max reps
